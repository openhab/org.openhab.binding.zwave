# ERC307 Z-Wave Plus Wireless Receiving Controller
This describes the Z-Wave device *ERC307*, manufactured by *[Everyoo](http://www.everyoo.com)* with the thing type UID of ```everyoo_erc307_00_000```.

The ERC307 supports routing. This allows the device to communicate using other routing enabled devices as intermediate routers.  This device is also able to participate in the routing of data between other devices in the mesh network.
