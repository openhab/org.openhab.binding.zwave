#### 2: Blink

This is a double byte value.  


##### Overview 

This is a double byte value. The LSB defines the total time the device need to blink. The value is set in seconds. The MSB defines the on/off interval of the blinking. The unit is 0.1 s.

Configuration Value 1 is to Specify the time that Micro Double SES need blink, The unit is Second; Configuration Value 2 is to Specify the Cycle of on/off, the unit of it is 0.1 second. For example: if we set Configuration Value 1 to '15',Configuration Value 2 to '10',then Micro Double SES will open 0.5 second, close 0.5 second, and repeat for 14 times.Â
